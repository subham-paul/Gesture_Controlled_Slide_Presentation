What is GestureSlides AI?

GestureSlides AI is an intelligent presentation controller that allows users to navigate slides using hand gestures. The system uses MediaPipe Hands, OpenCV, and Python to detect motion and convert gestures into keyboard commands like Next and Previous.

How It Works ⚙️

Camera Input

The webcam captures real-time hand movements.

Gesture Detection

MediaPipe detects hand landmarks and motion direction.

Slide Control

Keyboard events move slides forward or backward.
